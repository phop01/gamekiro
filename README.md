# 🦴 Multi-Person Pose Detection + Hand Skeleton

ระบบตรวจจับท่าทางร่างกายและนิ้วมือแบบ Real-time ผ่านกล้องเว็บ รองรับหลายคนพร้อมกัน

## ✨ ความสามารถ

- 🧍 ตรวจจับร่างกายทั้งตัว 33 จุด (หน้า, แขน, ลำตัว, ขา)
- 🖐️ ตรวจจับนิ้วมือละเอียด 21 จุดต่อมือ (ทุกข้อนิ้ว)
- 👥 รองรับหลายคนพร้อมกัน (สูงสุด 5 คน, 6 มือ)
- 🎨 แสดงโครงกระดูก (Skeleton) สีแยกตามคน
- 📊 แสดง FPS, จำนวนจุด, ส่วนร่างกายที่เห็น
- 🪞 กลับภาพซ้าย-ขวาได้
- 💰 ฟรี 100% ไม่มีค่าใช้จ่าย

## 🚀 วิธีใช้

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์ (Chrome/Edge แนะนำ)
2. อนุญาตกล้อง
3. ยืนให้กล้องเห็นร่างกาย
4. ระบบจะแสดงโครงกระดูกอัตโนมัติ

## 🎮 ปุ่มควบคุม

| ปุ่ม | หน้าที่ |
|------|---------|
| 🦴 โครงกระดูก | เปิด/ปิดเส้นโครงกระดูกร่างกาย |
| 🖐️ นิ้วมือละเอียด | เปิด/ปิดการตรวจจับนิ้วมือ |
| ⚪ จุดเท่านั้น | แสดงจุดอย่างเดียว ไม่มีเส้น |
| 🪞 กลับภาพ | กลับภาพซ้าย-ขวา |

## 🛠️ เทคโนโลยี

- [MediaPipe Tasks Vision](https://ai.google.dev/edge/mediapipe/solutions/vision/pose_landmarker) — Pose Landmarker (ร่างกาย)
- [MediaPipe Hand Landmarker](https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker) — Hand Landmarker (นิ้วมือ)
- HTML5 Canvas — วาดโครงกระดูก
- WebCamera API — เข้าถึงกล้อง

## 📋 ข้อกำหนด

- เบราว์เซอร์ที่รองรับ WebGL (Chrome, Edge, Firefox)
- กล้องเว็บ
- ไม่ต้องติดตั้งอะไรเพิ่ม ไม่ต้องมี server

## 📝 หมายเหตุ

- ทุกอย่างรันบนเครื่องผู้ใช้ (Client-side) ไม่ส่งข้อมูลไป server ใดๆ
- Model โหลดจาก Google CDN ครั้งแรกอาจใช้เวลาสักครู่
- ยิ่งเห็นร่างกายชัด ยิ่งตรวจจับได้แม่นยำ
