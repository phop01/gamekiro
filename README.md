# 🎮 เกมจับไอเท็มด้วยมือ (Hand Catch Game)

เกมเว็บไซต์ที่ใช้กล้องตรวจจับมือผ่าน AI เพื่อจับไอเท็มที่หล่นลงมาจากด้านบน

![Hand Catch Game](https://img.shields.io/badge/Game-Hand%20Catch-orange)
![TensorFlow.js](https://img.shields.io/badge/AI-TensorFlow.js-blue)
![HTML5](https://img.shields.io/badge/Tech-HTML5-red)

## 🎯 วิธีเล่น

1. เปิดเว็บไซต์และอนุญาตให้เข้าถึงกล้อง
2. รอระบบโหลด AI ตรวจจับมือ (ครั้งแรกอาจใช้เวลาสักครู่)
3. กดปุ่ม **เริ่มเกม**
4. ยกมือให้กล้องเห็นจนระบบตรวจจับได้ (จะเห็นโครงกระดูกมือ)
5. ใช้มือแตะไอเท็มที่หล่นลงมาเพื่อเก็บคะแนน
6. หลีกเลี่ยงไอเท็มที่ลดคะแนน!

## ⭐ ไอเท็มในเกม

### ✅ ไอเท็มเพิ่มคะแนน
| ไอเท็ม | คะแนน |
|--------|--------|
| ⭐ ดาว | +10 |
| 💎 เพชร | +20 |
| 🍎 แอปเปิ้ล | +5 |
| 💰 ถุงเงิน | +15 |

### ❌ ไอเท็มลดคะแนน
| ไอเท็ม | คะแนน |
|--------|--------|
| 💣 ระเบิด | -20 |
| ☠️ หัวกะโหลก | -15 |
| 🧊 น้ำแข็ง | -10 |

## 🛠️ เทคโนโลยีที่ใช้

- **TensorFlow.js** — รัน AI ตรวจจับมือบนเบราว์เซอร์
- **Handpose Model** — โมเดลตรวจจับมือ 21 จุด (landmarks)
- **WebGL** — เร่งความเร็วการประมวลผล AI ด้วย GPU
- **Canvas API** — วาดโครงกระดูกมือและไอเท็ม
- **getUserMedia API** — เข้าถึงกล้องเว็บแคม

## 🚀 วิธีรัน

### วิธีที่ 1: Live Server (แนะนำ)
1. ติดตั้ง extension **Live Server** ใน VS Code
2. คลิกขวาที่ `index.html` → **Open with Live Server**

### วิธีที่ 2: Python HTTP Server
```bash
cd d:\02
python -m http.server 8000
```
เปิดเบราว์เซอร์ไปที่ `http://localhost:8000`

### วิธีที่ 3: Node.js
```bash
npx serve d:\02
```

> ⚠️ **สำคัญ:** ต้องเปิดผ่าน `localhost` หรือ `HTTPS` เท่านั้น เพราะเบราว์เซอร์จำกัดการเข้าถึงกล้องบน HTTP ธรรมดา

## 📋 ความต้องการ

## 📝 หมายเหตุ

- ครั้งแรกที่เปิดจะใช้เวลาโหลด model ประมาณ 3-5 วินาที
- ครั้งต่อไปเบราว์เซอร์จะ cache model ไว้ทำให้โหลดเร็วขึ้น
- แนะนำให้มีแสงสว่างเพียงพอเพื่อให้ตรวจจับมือได้ดี
- คะแนนจะไม่ต่ำกว่า 0
